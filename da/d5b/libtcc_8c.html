◆ get_tok_str()

char* get_tok_str ( int  v,
CValue *  cv 
)

Get the string representation of a token.

The get_tok_str function returns the string representation of the token with the specified value v and associated CValue cv. The resulting string is stored in a static buffer.

Parameters
vThe token value.
cvThe associated CValue.
Returns
Pointer to the string representation of the token.
Note
The returned pointer is valid until the next call to get_tok_str.
